Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It describes the structure of a web page and consists of a series of elements that enclose different parts of the content to make it appear or behave in a certain way.

Basic HTML Elements

HTML elements are the building blocks of HTML pages. Here are some of the most commonly used elements:

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (link)
  • <img>: Image
  • <div>: Division or section
  • <span>: Inline container
